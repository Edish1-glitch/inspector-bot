# Inspector Reminder Bot 🤖

בוט טלגרם שמזכיר לך לסמן כניסה ויציאה ב-Inspector.

## הגדרת משתני סביבה ב-Railway

| משתנה | תיאור |
|-------|-------|
| `BOT_TOKEN` | הטוקן שקיבלת מ-BotFather |
| `USER_ID` | ה-ID שלך מ-userinfobot |

## שימוש בבוט

### הגדרת משמרות לשבוע:
```
/set_shifts ראשון:בוקר שני:צהריים שלישי:לילה שישי:כפולה_בוקר
```

### סוגי משמרות:
- `בוקר` — 07:00–15:00
- `צהריים` — 15:00–23:00
- `לילה` — 23:00–07:00
- `כפולה_בוקר` — 07:00–19:00
- `כפולה_לילה` — 19:00–07:00

### צפייה במשמרות מוגדרות:
```
/list_shifts
```

## איך עובד הבוט
- 5 דקות לפני תחילת משמרת — תזכורת לסמן כניסה
- 5 דקות לפני סוף משמרת — תזכורת לסמן יציאה
- אם לא אישרת — "נודניק" כל 10 דקות עד שתלחץ על הכפתור
